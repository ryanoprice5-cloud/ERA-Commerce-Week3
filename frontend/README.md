# ERA Commerce — Company Store Frontend

Frontend for ERA Academy Week 3 backend course.
Built with React + Vite + Tailwind CSS.

## Prerequisites
- Node.js installed
- Backend running on port 3000

## Setup
1. git clone [repo-url]
2. cd ERA-Commerce-Week3/frontend
3. npm install
4. cp .env.example .env
5. Confirm .env contains: VITE_API_URL=http://localhost:3000
6. npm run dev
7. Open browser at http://localhost:5173

## Notes
- Backend must be running before starting frontend
- Do not commit .env file
- Login creates a JWT token stored in localStorage
- Admin credentials required for reports and product management
